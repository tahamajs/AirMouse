# Video Script (max 5 minutes)

**0:00 - Intro**  
Show both devices, state your names and student IDs.

**0:30 - Calibration**  
Open app, enter IP, tap "Calibrate", show phone being still (gyro), then figure-8 pattern (mag). Wait for "Complete".

**1:15 - Connection**  
Tap "Start". Show Python server terminal (or GUI) indicating connection.

**1:45 - Cursor movement**  
Rotate phone around Z (left/right) → cursor moves horizontally. Rotate around X (up/down) → vertically. Show smooth tracking.

**2:30 - Click**  
Flick phone quickly around Y axis → cursor clicks on an icon or Notepad. Show that a character appears.

**3:15 - Scroll**  
Move phone linearly up/down along Y axis → page scrolls. Show both directions.

**4:00 - Drift test**  
Place phone on table, show cursor stays still for 10 seconds. (Prove no drift.)

**4:30 - Perfetto note**  
Briefly mention that you have answered the 11 questions using traces (show trace file if possible).

**4:50 - Outro**  
Thank the instructors, mention any challenges.
