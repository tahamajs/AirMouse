# Calibration UI

## Folder
`presentation/ui/calibration`

## Purpose
This folder presents the sensor calibration workflow used before control features unlock.

## Main files
- `CalibrationScreen.kt`
- `CalibrationViewModel.kt`
- `CalibrationResultScreen.kt`

## Responsibilities
- Guide the user through gyroscope, accelerometer, and magnetometer calibration.
- Display formulas and progress clearly.
- Store calibration results for later use.

## Demo expectation
- The page should look like a lab/demo screen and make the calibration sequence easy to follow.
