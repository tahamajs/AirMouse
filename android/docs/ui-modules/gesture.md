# Gesture UI

## Folder
`presentation/ui/gesture`

## Purpose
This folder provides gesture training and gesture studio screens.

## Main files
- `GestureStudioScreen.kt`
- `GestureStudioViewModel.kt`
- `GestureStudioState.kt`
- `GestureStudioEvent.kt`

## Responsibilities
- Let the user train or preview gesture behavior.
- Show waveform or live gesture feedback.
