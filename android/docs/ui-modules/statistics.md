# Statistics UI

## Folder
`presentation/ui/statistics`

## Purpose
This folder tracks usage and performance metrics.

## Main files
- `StatisticsScreen.kt`
- `StatisticsViewModelImpl.kt`
- `StatisticsUiState.kt`
- `StatisticsScreenEvents.kt`

## Responsibilities
- Show app usage, connection totals, calibration counts, and related metrics.
- Provide a visible summary for the demo and assignment report.
