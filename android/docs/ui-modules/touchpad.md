# Touchpad UI

## Folder
`presentation/ui/touchpad`

## Purpose
This folder contains the touchpad-style mouse control screen.

## Main files
- `TouchpadScreen.kt`
- `TouchpadViewModel.kt`
- `TouchpadState.kt`

## Responsibilities
- Turn finger input into pointer movement, taps, and gestures.
- Show touchpad-specific controls and mode information.
- Fit the app’s final demo style and connection gating.
