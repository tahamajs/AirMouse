# Logs UI

## Folder
`presentation/ui/logs`

## Purpose
This folder shows server and app activity logs.

## Main files
- `ServerLogsScreen.kt`
- `ServerLogsViewModel.kt`
- `ServerLogsState.kt`

## Responsibilities
- Display connection events, calibration messages, and debug output.
- Support reviewing what happened during pairing and control sessions.
- Help verify runtime behavior during the demo.
