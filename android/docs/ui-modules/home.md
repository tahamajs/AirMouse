# Home UI

## Folder
`presentation/ui/home`

## Purpose
The home folder is the control center of the app.

## Main files
- `HomeScreen.kt`
- `HomeViewModel.kt`
- `HomeActivity.kt`

## Responsibilities
- Show live sensor values and motion preview.
- Show connection status and pending/accepted/failed states.
- Gate control features until the user is registered and the device is calibrated.
- Provide quick access to the major modules in the app.

## Demo expectation
- This screen should clearly show that the app is ready, waiting, connected, or locked.
