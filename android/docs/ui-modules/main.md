# Main UI

## Folder
`presentation/ui/main`

## Purpose
This folder contains the app shell:
- `MainScreen` sets up the drawer, bottom bar, and overall scaffold.
- `MainNavHost` registers the app routes.
- `MainViewModel` coordinates top-level UI state.

## Notes
- The drawer acts as the primary navigation hub.
- The home screen is the default landing page.
- This layer connects the rest of the UI folders together.
